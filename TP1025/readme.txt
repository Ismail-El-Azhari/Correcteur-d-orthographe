Lors de l'exécution du programme. 

L'interface graphique est intuitive, il y a un espace pour écrire le texte. Vous ne pouvez vérifier un texte sans avoir ouvert un dictionnaire. Si vous essayez de corriger un texte vide après avoir ouvert un dictionnaire, vous aurez un message d'erreur.Vous pouvez importer un texte à partir du menu FILE , et ensuite OPEN permettant l'affichage d'un texte à partir d'un fichier. Vous pouvez sauvegarder le contenu de votre texte dans un fichier à partir de FILE , SAVE dans le répertoire désiré. Une fois le texte écrit et le dictionnaire chargés, le programme crée une liste de mot unique et compare cette liste avec la liste des mots créés à partir du fichier dictionnaire. Il fait un comparatif et crée un nouveau tableau contenant tous les mots qui ne sont pas dans le dictionnaire. Ces mots sont ensuite soulignés. Dans la fenêtre de droite, nous avons une suggestion de 5 mots. Cette fonction non fonctionnelle permet la suggestion des mots de l'index #1 de la liste des mots invalide. Mais ne permet pas le remplacement du texte, vous devez le faire de façon manuelle.
